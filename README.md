## :rocket: Rodando o projeto
Para rodar o repositório é necessário clonar o mesmo, dar o seguinte comando para iniciar o projeto:
```
git clone git@github.com:ur4sh1/Ignite-Feed.git
```
Instalar o npm
```
npm i
```
Executar
```
npm run dev
```
